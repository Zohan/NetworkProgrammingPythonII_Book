For loops

The for loop processes each item in a list. Each item in turn is (re-)assigned to the loop variable, and the body of the loop is executed.

Problem

A Domain Name Server converts site names, like google.com, to their IP address 172.217.3.174. Using a dictionary is faster than a list, because you do not need to check other elements to find the right one.

1. Write a fake_create_initial_network_list function, that creates list dict with 1000 random IPs.

2. Write a ip_in_network_list function, that takes a network list and an IP, and returns the IP if it was found in the last. Return "Not found" if name doesn't exist.
