17.6 Adding Other Methods

A method behaves like a function, but it is invoked on a specific instance. For example, with a server named linux, linux.set_server_ip('192.168.0.1') asks the server object to perform its set_server_ip method, and set the server_ip to 192.168.0.1. Methods are accessed using dot notation.

All methods defined in a class that operate on objects of that class will have self as their first parameter. This serves as reference to the object itself which in turn gives access to the state data inside the object.

Problem

Add a method, get_server_ip, that returns the object's server_ip.
